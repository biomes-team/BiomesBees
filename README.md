# Biomes! Bees

A simple and cozy mod that adds beekeeping to the Rim. Queen Bees can be found from wild hives in most biomes or purchased from traders. Queen bees can be used to build apiaries to produce honeycomb that can then be processed into various beekeeping based resources. You can now get your colonists drunk off mead or feed them entirely off honey!
